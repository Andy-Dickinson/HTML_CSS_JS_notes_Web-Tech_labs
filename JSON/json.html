Using JSON

Open developer tools console

Following is run from script

var j_str = '{"firstname":"Andy"}';

var j_obj = JSON.parse(j_str);

Both are loged to console

Adds element to JS object:
j_obj['lastname'] = 'Dickinson';

Convers JS object into JSON string:
var s = JSON.stringify(j_obj);
console.log("stringifyed JS object = ", s);
